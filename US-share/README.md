# US-share Skills

Personal Codex skills for U.S. equity and ETF workflows.

## Skills

- `us-market-macro-regime`: Generate the daily U.S. macro trading-environment report with rates, volatility, credit, sentiment, event risk, and optional portfolio linkage.
- `lei-invest`: Maintain and apply an evidence-backed LEI/TheMarketMemo U.S. equity investing philosophy library from daily Gmail reports.
- `weekly-momentum-report`: Generate and deliver the weekly U.S. market momentum report from TheMarketMemo Patreon source, then archive the completed report in this repository.
- `tqqq-long-term-rolling-strategy`: Apply and backtest TheMarketMemo's TQQQ long-term rolling strategy with TQQQ/JEPQ/SGOV, QQQ 200MA, BTD, hedge, and rebalance rules.
