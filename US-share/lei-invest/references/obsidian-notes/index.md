# Obsidian Note Extracts

This directory stores compact extracts from local Obsidian notes related to LEI / TheMarketMemo.

Each extract should include:

- vault-relative source path
- file modified time and content checksum
- source URL or original post ID when available
- note type
- key source quotes or evidence
- extracted investment views
- changes made to `references/lei-investment-principles.md`

Write extracts in Chinese by default while preserving original source titles, URLs, tickers, and post IDs exactly.
