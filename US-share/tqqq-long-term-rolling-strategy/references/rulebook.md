# TQQQ 槓鈴策略规则

来源：TheMarketMemo Patreon 文章《TQQQ 槓鈴策略》，发布日期 2025-12-16；页面截至 2026-07-10 收盘披露该策略组合当年回报率为 30.48%。文章链接：<https://www.patreon.com/TheMarketMemo/posts/tqqq-gang-ling-146010736>

本文档用于把原文策略转成可执行、可监控、可回测、可复查的规则。它是研究记录，不构成投资建议。

## 1. 标的与角色

| 标的 | Futu 代码 | 角色 |
|---|---|---|
| TQQQ | US.TQQQ | 三倍纳斯达克 100 多头，核心驱动引擎 |
| JEPQ | US.JEPQ | Covered Call 现金收益与降波缓冲器 |
| JAAA | US.JAAA | AAA CLO ETF，绝对防御底仓与现金储备 |
| SGOV | US.SGOV | 可作为现金/短债替代观察标的，但不是原文默认底仓 |
| QQQ | US.QQQ | 原生纳斯达克 100 ETF，用于回撤、趋势、期权和对冲判断 |
| VIX | 可用指数/代理 | 黑天鹅保险触发和平仓指标 |
| NQ/MNQ | 期货执行工具 | 可选的纳指期货对冲工具 |
| QQQ Put/Call | 期权执行工具 | 可选的黑天鹅保险与凸性增持工具 |

## 2. 核心资产架构

默认目标为 50/25/25：

| 标的 | 目标权重 |
|---|---:|
| TQQQ | 50% |
| JEPQ | 25% |
| JAAA | 25% |

JAAA 可以被替换为 AAA 评级且派息率大于 4% 的其他标的。替代时必须明确写出替代品、评级/信用质量、派息率、流动性和与 JAAA 的差异。

策略目标是在长期做多 TQQQ 的基础上，用 JEPQ 权利金现金流与 JAAA 防守底仓平衡波动，并把极端情绪转化为下跌加仓或保险现金。

## 3. 上涨再平衡

触发条件：持续上涨行情中，TQQQ 部位触及组合总市值的 65%。

执行方式：将组合恢复到初始架构：

- TQQQ 降回 50%。
- JEPQ 回到 25%。
- JAAA 或合格现金替代品回到 25%。

回测与监控口径：没有可核验持仓时，不推断用户真实权重；只在模型组合或用户提供持仓可计算时触发。

## 4. 下跌增持 BTD

触发基准：QQQ 从最近一个历史高点的回撤，而不是 TQQQ 自身回撤。

动用现金储备分批买入 TQQQ：

| QQQ 从最近历史高点回撤 | 买入 TQQQ 的现金储备比例 |
|---:|---:|
| -12% | 现金储备的 1/3 |
| 超过 -20% | 剩余现金储备的 1/2 |
| 超过 -30% | 全部剩余现金储备 |

现金储备包括 JAAA、合格短债替代品和账面现金。若需要动用 JAAA，应先说明卖出 JAAA 后买入 TQQQ 的金额与剩余防守垫。

## 5. 黑天鹅保险（非必须）

预算：账户总市值的 1%-2%，视为常规保险费支出。

触发条件：VIX 接近 12，处于历史低位区间。

执行方式：

- 买入约 1 年 DTE 的 QQQ Put。
- Delta 介于 -0.10 到 -0.15。

动态展期：

- 当剩余 DTE < 180 天，且 VIX 仍处于低位时，平仓残值并重新买入新的一年期合约。

平仓兌现：

- VIX 突破 35，或合约价值增长 300% 时平仓。
- 获得现金补偿后进入现金库，用于后续 BTD 增持 TQQQ。

监控口径：没有期权链和真实持仓时，只发送“保险检查提醒”，不估算收益，不把保险建议写成必做动作。

## 6. 股指期货对冲（非必须）

目的：避免长期熊市中 TQQQ 的灾难性损失。

对冲基础：以原生指数 QQQ 的趋势推演判断上升趋势是否即将终结。

执行方式：

1. 计算账户总体 beta。
2. 应对冲市值 = 账户市值 x beta。
3. 应持有期货合约数量 = 应对冲市值 / （纳指期货价格 x 合约乘数）。
4. 经趋势推演判断下跌趋势即将终结时平仓。

监控口径：趋势推演需要独立技术判断或可复现模型；不能仅凭单日跌幅自动开空。

## 7. 凸性增持策略（非必须）

目标：用小额固定亏损换取不对称反弹收益。

资金来源：JEPQ 与 JAAA 产生的配息分红。

执行条件：QQQ 满足“中长期多头 + 短期回撤”的趋势特征。

执行方式：

- 买入 QQQ Call。
- DTE 30-45 天。
- 轻微 OTM。
- Delta 0.4-0.5。

止盈：

- 期权价值增长 100% 时平仓一半。
- 剩余部位在获利 +200%，或 QQQ 出现上涨乏力时全数平仓。

止损：

- 期权价值下跌 -50% 无条件止损。

监控口径：没有期权链、期权持仓和用户许可时，只发送“凸性增持检查提醒”，不估算期权收益，不自动筛合约下单。

## 8. 回测默认假设

1. 初始资金按 TQQQ/JEPQ/JAAA = 50%/25%/25% 建仓。
2. 日线收盘价执行，允许小数股。
3. 不计佣金、滑点、税费、融资利息、期货保证金收益和期权买卖价差。
4. JEPQ 与 JAAA 分红按派息日进入现金储备。
5. BTD 使用 QQQ 从最近历史高点的回撤触发。
6. 再平衡在 TQQQ 权重达到 65% 时回到 50/25/25。
7. VIX、QQQ Put、QQQ Call、NQ/MNQ 只在有可复现历史数据或真实持仓明细时量化。
